Excercise in parsing syntax trees.

Find the maximum element in the tree. 
You must print the maximum element in the following format: Max: XX
Where XX is the maximum element's value (an integer).

// Valid tree inputs have the following format:
//
// (1 (2 (3)) (4) (5 (6) (7)))
//
//   which produces
//
//     1
//    /|\
//   2 4 5
//   |  / \
//   3 6   7

FYI An abstract syntax tree (AST), or just syntax tree, is a tree representation of the abstract syntactic structure of source code written in a programming language. Each node of the tree denotes a construct occurring in the source code.
